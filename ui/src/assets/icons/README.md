# Icon Usage Convention

This folder stores UI icon assets used by the app.

## Current Canonical Info Icon

- `info-icon.svg` is the canonical icon for relevance/tooltip info indicators.

## Rules

- Keep one canonical icon per semantic purpose (avoid duplicates with different naming styles).
- Use kebab-case naming for new icons (example: `warning-icon.svg`).
- Prefer neutral-tone defaults unless a feature explicitly requires brand-color emphasis.
- Remove unused icon variants when a canonical icon is selected.
